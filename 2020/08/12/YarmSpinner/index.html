安装

安装有四种方式：

  • 通过 OpenUPM 来安装，是最推荐的，安装了 OpenUPM cli 之后通过一条命令即可完成安装。
  • 通过 Package Manager 的 Add package from git URL 来安装，这里的链接按照 Unity Manual 上的应该是可以使用 Http 或者 SSH 形式的，但我试了一次没有成功，因为上一种方法已经装好了，就没再进行尝试。
  • 通过修改 manifest.json 来安装，没试过，但在使用了 OpenUPM 的方式之后会发现在 manifest.json 中会出现差不多的配置。
  • 下载 .unitypackage 文件，没试过。

看示例

示例 是一个简单的场景，里面有两个 NPC，需要与其进行对话。如果是安装为 Package 的话，在 Package Manager 里面找到 YarmSpinner ，然后安装 Space，即可在找到名为 Space 的示例，如果是直接安装的 .unitypackage 文件的话，那可以直接找到 Space 的文件夹。
